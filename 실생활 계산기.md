# 📱 Project: 수학 기초 계산기 추가 및 실생활 일반 계산기 UI 개발 지시서

## 🎯 목표

1. `calculatorRegistry`에 [수학 기초] 카테고리에 해당하는 필수 공식(비례식, 백분율 등)을 추가합니다.
2. [실생활 계산기] 카테고리를 눌렀을 때는 기존의 수식 입력 폼이 아닌, 아이폰/갤럭시 기본 앱과 같은 형태의 **'일반 디지털 계산기(Standard Calculator) UI'** 컴포넌트가 화면에 렌더링되도록 구조를 분기 처리합니다.

---

## 1. [수학 기초] 카테고리 공식 데이터 (Data Registry)

기존 `calculatorRegistry` 배열에 아래의 수학 기초 항목들을 추가해 주세요.

```javascript
[
  {
    id: "math-proportion", category: "수학 기초", name: "비례식 계산 (A : B = C : X)",
    inputs: [{ id: "A", label: "A 값", unit: "" }, { id: "B", label: "B 값", unit: "" }, { id: "C", label: "C 값", unit: "" }],
    formulaLatex: "X = \\frac{B \\times C}{A}",
    calculate: (A, B, C) => (B * C) / A // 결과: X의 값
  },
  {
    id: "math-percentage", category: "수학 기초", name: "백분율 계산 (A의 X%)",
    inputs: [{ id: "target", label: "전체 값 (A)", unit: "" }, { id: "percent", label: "퍼센트 (X)", unit: "%" }],
    formulaLatex: "\\text{Result} = A \\times \\frac{X}{100}",
    calculate: (target, percent) => target * (percent / 100)
  },
  {
    id: "math-percent-change", category: "수학 기초", name: "증감율 계산 (A에서 B로)",
    inputs: [{ id: "oldValue", label: "이전 값 (A)", unit: "" }, { id: "newValue", label: "새로운 값 (B)", unit: "" }],
    formulaLatex: "\\text{Change}(\\%) = \\frac{B - A}{A} \\times 100",
    calculate: (oldValue, newValue) => ((newValue - oldValue) / oldValue) * 100
  }
]
```

// 의사 코드(Pseudo-code) 예시
if (selectedCalculator.category === "실생활") {
  return `<StandardCalculator />`; // 일반 키패드 계산기 화면 렌더링
} else {
  return `<DynamicFormulaForm data={selectedCalculator} />`; // 기존의 수식 입력 폼 렌더링
}

* 실생활 계산기를 눌렀을 때 다른 것 처럼 계산 카테고리가 뜨는게 아닌 일단 계산기 UI가 뜨고 계산할 수 있도록 구현할 것 이다.
* 계산기 UI에 나오는 기본적인 사칙연산을 수행 할 수 있도록 하게 만들어 주길 원함.
