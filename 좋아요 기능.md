# 좋아요 싫어요 누르고 저장될 수 있도록 하는 구현계흭서

**수정사항:** 지금 현재 커뮤니티창 글 내에서 좋아요 Like 이모지가 담긴 네모 칸을 누르면 +1이 되고 다시 누르면 -1이 되게 구현은 되어 있지만 이게 데이터로써 저장되고 처리되지 않고, 그냥 다시 나오게 되면 메인 화면에 있는 Like 이모지와 커뮤니티 글 목록 창에 있는 Like 이모지는 반영이 되어지지 않음. 그래서 눌렀을 때 메인 화면과 커뮤니티 글 목록창에서 +1과 -1이 돼서 얼마나 누가 좋아요를 많이 누르는지 파악하고 그거에 따라 인기글을 좋아요 LIke 이모지 순으로 정렬 할 수 있게끔 메인화면에 송출 되도록 해야함.
