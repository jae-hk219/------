
# 🔥 Project: 보일러 및 증기 시스템(Steam System) 전문 계산기 로직 개발 지시서

## 🎯 목표

열역학 기반의 보일러 효율, 스팀/열량 단가, 그리고 증기의 건도(Dryness) 및 에어 혼입에 관련된 전문 계산기 모듈을 구현합니다. 사용자가 입력한 열량 및 압력 데이터를 바탕으로 복잡한 공학 수식을 연산하고, `formulaLatex`를 통해 시각적으로 신뢰성을 제공해야 합니다.

## 📋 공식 데이터 명세 (Boiler & Steam Calculations)

아래의 JSON 구조를 `calculatorRegistry` 배열에 추가하여 동적 폼을 구성하세요.

```javascript
[
  // ==========================================
  // 1. 보일러 및 비용 분석 (Efficiency & Cost)
  // ==========================================
  {
    id: "boiler-efficiency", category: "보일러 계산", name: "보일러 효율",
    inputs: [{ id: "Gs", label: "실제 증발량", unit: "kg/h" }, { id: "h2", label: "발생증기 엔탈피", unit: "kJ/kg" }, { id: "h1", label: "급수 엔탈피", unit: "kJ/kg" }, { id: "B", label: "연료 소비량", unit: "kg/h" }, { id: "Hl", label: "연료 저위발열량", unit: "kJ/kg" }],
    formulaLatex: "\\eta = \\frac{G_s(h_2 - h_1)}{B \\times H_l} \\times 100",
    calculate: (Gs, h2, h1, B, Hl) => ((Gs * (h2 - h1)) / (B * Hl)) * 100 // 결과: %
  },
  {
    id: "heat-unit-cost", category: "비용 분석", name: "열량 단가",
    inputs: [{ id: "Pf", label: "연료 단가", unit: "원/kg" }, { id: "Hl", label: "저위발열량", unit: "kcal/kg" }, { id: "eff", label: "보일러 효율", unit: "%" }],
    formulaLatex: "\\text{Cost}_{heat} = \\frac{P_f}{H_l \\times (\\eta / 100)} \\times 10^4",
    calculate: (Pf, Hl, eff) => (Pf / (Hl * (eff / 100))) * 10000 // 결과: 원/10,000kcal 기준
  },
  {
    id: "steam-unit-cost", category: "비용 분석", name: "증기 단가 (Steam Cost)",
    inputs: [{ id: "Pf", label: "연료 단가", unit: "원/kg" }, { id: "h2", label: "증기 엔탈피", unit: "kcal/kg" }, { id: "h1", label: "급수 엔탈피", unit: "kcal/kg" }, { id: "Hl", label: "저위발열량", unit: "kcal/kg" }, { id: "eff", label: "보일러 효율", unit: "%" }],
    formulaLatex: "\\text{Cost}_{steam} = \\frac{P_f \\times (h_2 - h_1)}{H_l \\times (\\eta / 100)}",
    calculate: (Pf, h2, h1, Hl, eff) => (Pf * (h2 - h1)) / (Hl * (eff / 100)) // 결과: 원/kg
  },

  // ==========================================
  // 2. 증기 건도 및 감압 (Steam Dryness & Depressurization)
  // 등엔탈피(Isenthalpic) 팽창 원리 적용
  // ==========================================
  {
    id: "dryness-depressurization", category: "증기 상태", name: "감압에 따른 건도 개선",
    inputs: [{ id: "x1", label: "감압 전 건도", unit: "소수점" }, { id: "hf1", label: "감압전 현열", unit: "kJ/kg" }, { id: "hfg1", label: "감압전 잠열", unit: "kJ/kg" }, { id: "hf2", label: "감압후 현열", unit: "kJ/kg" }, { id: "hfg2", label: "감압후 잠열", unit: "kJ/kg" }],
    formulaLatex: "x_2 = \\frac{h_{f1} + x_1 h_{fg1} - h_{f2}}{h_{fg2}}",
    calculate: (x1, hf1, hfg1, hf2, hfg2) => ((hf1 + (x1 * hfg1)) - hf2) / hfg2 // 결과: 감압 후 건도 (x2)
  },
  {
    id: "dryness-separation", category: "증기 상태", name: "응축수 분리 및 감압 후 건도",
    inputs: [{ id: "hf1", label: "분리후/감압전 현열", unit: "kJ/kg" }, { id: "hfg1", label: "분리후/감압전 잠열", unit: "kJ/kg" }, { id: "hf2", label: "감압후 현열", unit: "kJ/kg" }, { id: "hfg2", label: "감압후 잠열", unit: "kJ/kg" }],
    formulaLatex: "x_2 = \\frac{h_{f1} + 1.0 \\times h_{fg1} - h_{f2}}{h_{fg2}}",
    calculate: (hf1, hfg1, hf2, hfg2) => ((hf1 + hfg1) - hf2) / hfg2 
    // AI 참고: 응축수 분리기(Separator) 통과 직후는 건도(x1)를 1.0(100%)으로 가정한 후 감압(등엔탈피) 계산.
  },

  // ==========================================
  // 3. 에어 혼입 및 단열 (Air Mixing & Insulation)
  // 돌턴의 분압 법칙(Dalton's Law) 적용
  // ==========================================
  {
    id: "air-volume-ratio", category: "증기 상태", name: "에어체적비",
    inputs: [{ id: "Pt", label: "전체 혼합 압력", unit: "MPa" }, { id: "Ps", label: "온도에 해당하는 증기 분압", unit: "MPa" }],
    formulaLatex: "V_{air}(\\%) = \\frac{P_t - P_s}{P_t} \\times 100",
    calculate: (Pt, Ps) => ((Pt - Ps) / Pt) * 100 // 결과: 공기가 차지하는 체적비(%)
  },
  {
    id: "insulation-thickness", category: "배관설계", name: "경제적 보온의 두께 (단순화 모델)",
    inputs: [{ id: "k", label: "보온재 열전도율", unit: "W/m·K" }, { id: "Ti", label: "내부 온도", unit: "℃" }, { id: "Ta", label: "외기 온도", unit: "℃" }, { id: "r1", label: "배관 외반경", unit: "m" }, { id: "Q_allow", label: "허용 열손실량", unit: "W/m" }],
    formulaLatex: "r_2 = r_1 \\times \\exp\\left(\\frac{2\\pi k(T_i - T_a)}{Q_{allow}}\\right)",
    calculate: (k, Ti, Ta, r1, Q_allow) => r1 * Math.exp((2 * Math.PI * k * (Ti - Ta)) / Q_allow)
    // 결과: 보온재 포함 총 반경(m). 두께 = r2 - r1
  }
]
```


* 여기 역시 카테고리에 있는 이름은 제외하고 name: 뒤에 있는 이름은 배관공사 카테고리에 넣어서 계산 진행 할 수 있도록 할 것이다.
* 여기 모든 것들을 다 참고하려고 하지말고 공식위주로 분석한 다음 앱에 맞도록 공식을 코딩해 수식을 집어넣어 계산 할 수 있도록 할 것.
